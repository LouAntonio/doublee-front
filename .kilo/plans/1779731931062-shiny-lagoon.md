# Refactor Plan: Admin Dashboard (/dbe) to Match /sobre Aesthetics

## Current State Analysis

### `/sobre` Page Design Language
- **Typography**: `font-display` (Fredoka) for headings, `font-body` (Nunito) for body text
- **Colors**: 
  - Background: `bg-sand` (#F5F0EB) - warm beige
  - Primary accent: `text-accent` (#F97316) - orange
  - Text: `text-[#1C1917]` (dark brown), `text-[#78716C]` (warm gray)
- **Animations**: Staggered `animate-fade-in-up` with delays (0.1s, 0.25s, 0.4s, etc.)
- **Decorative elements**: `geo-pattern` background overlay
- **UI patterns**: Rounded cards (`rounded-3xl`), subtle shadows, warm color scheme

### Current `/dbe` Admin Pages
- **Typography**: Same fonts but styled differently
- **Colors**: Cool slate palette (grays, blues) - dark sidebar with light content
- **Background**: `bg-slate-50` with gradient overlays
- **Style**: Dark sidebar, different card styling

## Refactoring Goals

Apply the warm, organic aesthetic from `/sobre` to all `/dbe` admin pages for visual consistency:

### 1. Color Palette Changes
- Replace `bg-slate-50` → `bg-sand`
- Replace `text-slate-*` → `text-[#1C1917]` and `text-[#78716C]`
- Replace cool status colors (blue, indigo, rose) → warm variants using `accent` family
- Status badges: emerald/amber/rose → warmer greens/oranges/raspberry tones

### 2. Layout Changes
- **Sidebar**: Change from dark to warm off-white/sand with accent borders
- **Cards**: Use `rounded-3xl` instead of `rounded-2xl`, warmer shadows
- **Background**: Add `geo-pattern` overlay for consistency

### 3. Component Updates
- **AdminDashboard.jsx**: Update stat cards to use accent colors, warm backgrounds
- **AdminUsers.jsx**: Update table styling, badges, buttons to warm palette
- **AdminStores.jsx**: Update modal, table, and all interactive elements
- **AdminDashboardLayout.jsx**: Complete sidebar and header restyling

### 4. Animation Enhancements
- Add staggered fade-in animations to all page elements
- Use consistent animation delays matching `/sobre` pattern (0.1s, 0.25s, 0.4s, etc.)
